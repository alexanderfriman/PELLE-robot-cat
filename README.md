# PELLE-robot-cat
This repository contains documentation, code and 3D files for PELLE a walking robot cat. PELLE (Playstation-Encoded-Lightweight-Legged-Entity) is a group project for the Mechatronic exercises course at Aalto university and was presented at the Mechatronics Circus 2026.

Documentation available at: [https://alexanderfriman.github.io/PELLE-robot-cat/PELLE_Robot_Cat.htm](https://alexanderfriman.github.io/PELLE-robot-cat/PELLE_Robot_Cat.htm)
